What is aria-owns?

aria-owns is an attribute that allows to define the parental relationship of an object in the Accessibility Object Model.

How to use aria-owns?

The attribute accepts a list of id separated by a space character. If an element has aria-owns and DOM children, the order of the child elements would be:
  1. The actual DOM children.
  2. The referenced children by id in aria-owns.

When can I use aria-owns?

It is useful when the parent/child relationship cannot be determined from the DOM.
